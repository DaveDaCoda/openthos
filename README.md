# openthos

## Cloning wikis locally to your computer
```
git clone https://github.com/openthos/openthos.wiki.git
or 
git clone git@github.com:openthos/openthos.git
```

## Adding wiki page
```
git add filename.md
git commit -m "comment"
```

## Removing wiki page
```
git rm filename.md
git commit -m "comment"
```
## Push or pull
```
git push
or
git pull
```
